# ICyberSol Landing Page

This repo contains a single-file static landing page for **ICyberSol**.

## Files

- `index.html` - complete landing page with inline CSS and no build step

## Host It Anywhere

Because this is plain static HTML, you can host it with:

- GitHub Pages
- Netlify
- Vercel static hosting
- Cloudflare Pages
- Any Apache or Nginx server

## Local Preview

Open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize

- Replace `hello@icybersol.com` with your real contact email
- Update text to match your exact product claims
- Add a logo or product screenshots later if needed
